# מחנה קיץ בעברית

אתר סטטי פשוט לקורס קיץ בעברית.

## איך מפעילים
1. חלץ את קובץ ה-ZIP.
2. פתח את הקובץ `index.html` בדפדפן.
3. בחר שיעור וענה על הבוחן.

## איך מוסיפים שיעורים
פתח את `lessons.js` והוסף אובייקט חדש בתוך המערך `LESSONS`.

האתר תומך ב:
- התאמת מילים עברית–אנגלית
- טקסט קריאה
- שאלות אמריקאיות
- שאלה פתוחה קצרה עם בדיקת מילות מפתח בסיסית
- דקדוק
- Cloze
- שמירת ציונים בדפדפן
